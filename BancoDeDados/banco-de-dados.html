O que é banco de dados?

Todo lugar utilizado para armazenamento de informações, sejo do seu sistema, como dos usuários que o acessam.

  • Bloco de notas
  • Caixa de arquivo com fichas

Tipos de bancos de dados

  • Banco de Dados Relacional;
    • conceito de tabelas;
    • relação entre tabelas

    Os bancos de dados relacionais conhecidos são:

    • MySQL
    • PostgreeSQL
    • SQLite
    • Oracle database
    • Microsoft SQL Server
  • Banco de Dados não-relacional;
    • Banco de dados com outros tipos de conceitos
    • Gráfico
    • Documento
    • chave-valor(key-value)

    Os bancos de dados não-relacionais conhecidos são:

    • mongoDB
    • redis
    • cassandra
    • neo4j
    • amazon DyanmoDB
    • influxdb

Inserts

Utilizado para inserir dados nas tabelas.

INSERT INTO "users" (name, last_name, age)

VALUES ("João", "Francisco", 21)

Selects

Utilizado para consultas as informações do banco de dados.

SELECT *

FROM "users"


SELECT name, last_name

FROM "users"


Para fazer filtros

SELECT name, last_name

FROM "users"

WHERE age > 31

Update

UPDATE "users"

SET last_name = "Almeida"

WHERE id = 3;

Delete

DELETE FROM "users"

WHERE id=4

Inner Join

Utilizado para relacionamento entre tabelas.

É a JUNÇÃP de duas tabelas.

Traz os dados que possuem alguma relação.

SELECT *

FROM "users"

INNER JOIN "users-logins" ON users-login.id_users = users.id
