O que é banco de dados?

Todo lugar utilizado para armazenamento de informações, sejo do seu sistema, como dos usuários que o acessam.

  • Bloco de notas
  • Caixa de arquivo com fichas

Tipos de bancos de dados

  • Banco de Dados Relacional;
    • conceito de tabelas;
    • relação entre tabelas

    Os bancos de dados relacionais conhecidos são:

    • MySQL
    • PostgreeSQL
    • SQLite
    • Oracle database
    • Microsoft SQL Server
  • Banco de Dados não-relacional;
    • Banco de dados com outros tipos de conceitos
    • Gráfico
    • Documento
    • chave-valor(key-value)

    Os bancos de dados não-relacionais conhecidos são:

    • mongoDB
    • redis
    • cassandra
    • neo4j
    • amazon DyanmoDB
    • influxdb
