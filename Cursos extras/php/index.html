Acessar:

  1. Meu computador
  2. wamp64
  3. www
  4. Criar uma nova pasta

Criar um virtual hosts

  1. Acessar bi, dentro de wamp64. acessar apache, apache2.4.23, conf
  2. Acessar o arquivo httpd.conf e editar com notepad++ e descomentar a linha de http-vhosts
  3. Acessar extra, httpd-vhosts.conf
  4. editar para adaptar ao projeto
  5. alteração: nome do projeto e caminho so projeto.
  6. para finalizar, acessar o arquivo de host do windows.
  7. Disco C, Windows,System 32, drivers, etc, hosts como administrador
  8. Acrescentar um localhost

  9. Atualizar o WampServer
  10. O projeto criado irá aparecer em YOUR PROJECTS
