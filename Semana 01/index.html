Conceitos
html
Envolve todo o documento html.
Doctype HTML
Diz que o documento irá trabalhar no HTML5
head
São elementos não visuais (com exceção do title), metadados e importações de arquivos. Envolve o cabeçalho do documento.
meta
Fornece as informações gerais sobre o documento.
body
Onde se encontra o projeto. Tags básicas, tags semanticas e afins.
