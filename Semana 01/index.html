Criando Tabelas

Para criar a tabela, você deverá utilizar

  1. tag table - representando uma estrutura
  2. tag thead - estrutura do cabeçalho
  3. tag tbody - estrutura do corpo
  4. tag tr - table row - representando a linha
  5. tag th - table head

Conhecendo os Forms

Forms são campos que recebem as entradas dos usuários.

O form é constituido de:

  1. Tag Form
  2. Tag input - é a tag de entrada
  3. Tag submit - é o botão que envia as informações para o servidor
