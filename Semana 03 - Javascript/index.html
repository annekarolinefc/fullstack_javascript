Javascript

Javascript

  • Nasceu em 1995
  • Criado por Brendan Eich
  • Surgiu para dar mais dinamismo às páginas HTML, visto que apenas CSS e HTML deixavam as páginas estáticas.
  • Usado principalmente para criar interfaces mais amigáveis.
  • Linguagem de programação executada diretamente no navegador.
  • Possui múltiplos paradigmas: Orientada a objeto funcional, orientada a eventos
  • Possui tipagem dinâmica e fraca.
  • Possui uma grande quantidade de bibliotecas que auxiliam durante o desenvolvimento.

São expressões construídas a apartir de operações com valores booleanos TRUE ou FALSE.

Também retornam um valor booleano.

Seguimos os resultados lógicos contidos na tabela verdade. A tabela verdade é resultado da combinação dos dois valores booleanos em todas as combinações possiveis.

As operações básicas que temos na programação dão E(and), OU(or) e NÃO(not).

let velocity = 80;

console.log("velocity (> maior) 6 || velocity (menor) 70") V ou F = V console.log(!(velocity>4))inverte os valores: NÃO F -> vira verdadeiro (V)

Condicional

Condicionais - if/else statements

São blocos de códigos que são executados baseado no resultado d euma expressão lógica.

Também são chamados de Controle de Fluxo.

Podem direcionar a aplicação por um determinado fluxo com base no valor das expressões.

If condition{...} else {...}

let velocity = 80;

if velocity < 40 {

console.log("Velocidade baixa")

}else{

console.log("Entrando em velocidade de risco.") }

Condicionais - switch

Estrutura de Controle de Fluxo.

Avalia uma expressão.

Procura a mlehor cláusula (caso) que atende ao resultado.

As cláusulas, ou casos são chamados de cases.

let velocity = 80

switch (velocity * 2) {

case 100:

console.log("Sua velocidade é 100km/s")

break

case 160:

console.log("Sua velocidade é 160km/s")

break

default:

console.log("Velocidade não identificada.")

}

(...)

Operador de agrupamento! As operações netre colchetes sao executadas mais cedo do que aquelas de fora.

    Métodos do Array

    Concat()

    Utilizado para unir vários arrays em um só.

    var fruit1 = ["banana", "apple", "pear"];

    var fruit2 = ["orange", "stranberry"];

    var totalFruits = fruit1.concat(fruit2);

    console.log(`Array de frutas 01: ${fruit1}`);

    console.log(`Array de frutas 02: ${fruit2}`);

    console.log(totalFruits);

    indexOf()

    Retorna o tipo primitivo do objeto especificado

    var Totalfruits = ["banana", "apple", "pear"];

    console.log(totalFruits.indexOf("apple"))

    Neste caso, retorna 1, pois maça esta no indice 01.

    Join()

    O método join() junta todos os elementos de um array em uma string, e retorna uma string.

    Permite que você defina o tipo de separador através de [separador = " ... "]

    lastIndexOf()

    Mostra a ultima posição do caractere que desejamos buscar.

    console.log("A ultima posição de r é: " + "palavra".lastIndexOf("r"))

    pop()

    Remove o ultimo elemento do array.

    var numeros = [1, 2, 3, 4]

    numeros.pop()

    console.log(numeros)

    push()

    Adiciona um elemento ao final do array

    numeros.push(4)

    console.log(numeros)

    reverse()

    Classifica os elementos em ordem decrescente.

    numeros.reverse()

    console.log(numeros)

    shift()

    Remove o primeiro elementos de um array.

    const array1 = [1, 2, 3];

    const firstElement = array1.shift();

    console.log(array1);

    console.log(firstElement);

    slice()

    Cria um novo array através de um array ja existente, isto é, faz uma copia de parte de um array.

    var frutasSupermercado = ["banana", "laranja", "limão", "maça"]

    var frutasCitricas = frutasSupermercado.slice(1, 3)

    console.log("As frutas citricas são: " + frutasCitricas)

    sort()

    Classifica os elementos em ordem alfabetica

    var alfabeto = ["b", "d", "a", "w", "m", "c", "y"]

    console.log("A ordem será: " + alfabeto.sort())

    splice()

    Adiciona um elemento na posição especificada podendo remover (ou não) elementos antigos

    frutasSupermercado = ["banana", "laranja", "limão", "maça"]

    frutasSupermercado.splice(4, 0, "maracujá")

    console.log("Somente adicionei um elemento na posição 04: " + frutasSupermercado);

    frutasSupermercado.splice(0, 1, "Abacaxi")

    console.log("Adicionei um elemento na posiçao 0 e removi um elemento: " + frutasSupermercado)

    toString()

    Converte elementos para String

    unshift()

    Adiciona um ou mais elementos no incio de um array e retorna o numero de elementos

    var arr = [1, 2];

    arr.unshift(0); // result of call is 3, the new array length

    // arr is [0, 1, 2]

    arr.unshift(-2, -1); // = 5

    // arr is [-2, -1, 0, 1, 2]

    arr.unshift([-3]); // arr is [[-3], -2, -1, 0, 1, 2]

    valueOf()

    Retorna o valor primitido de um objeto string

    var str = "Hello World!"

    console.log(str.valueOf())

    charAt()

    Concatena(junta) duas ou mais strings em uma.

    Recupera as correspondências de uma string em relação a um padrão de pesquisa.
