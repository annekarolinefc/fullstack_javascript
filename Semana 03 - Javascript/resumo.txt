Onde colocar o código javascript
    - Colocar dentro da tag script, dentro do body
    <body>
        <script></script>
    </body>

    - colocar um arquivo externo script.js 
    <link rel="stylesheet" href="style.css">

    - colocar a tag script dentro do head 
    <head>
        <script src="script.js"></script>
    </head>
