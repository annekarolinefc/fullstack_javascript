BABEL

Babel é uma biblioteca que basicamente converte o javascript moderno para javascript antigo aumentando as chances de que a sua aplicação seja compativel com browsers antigos.

A intalação do babel usando o npm é atraves de: npm install --save-dev @babel/core babel-loader @babel/present-env @babel/present-react

Transpilação com Babel

Transpilar é converter o código que está numa linguagem para outra.

Babel é utilizado para isso.

Desenvolvido em Javascript.

Utilizado para permitir compatibilidade com vageadores que ainda não suportam determinada versão.

babel.config.js

é um arquivo Javascript.

Configuramos os presets, plugins e etc.

Lido no momento de converter os arquivos.

é utilizado na raix do projeto para as configurações padrões.

.babelrc

é um arquivo JSON.

Configuramos os presets, plugins e etc.

Lido no momento de converter os arquivos.

é utilizado em subdiretórios para sobrescrever os valores padrões que vem no babel.config.js

Dia 02

Webpack

Conhecida como static module bundler.

Lê todo o código e monta um gráfico de dependências.

Este gráfico é utilizado para montar os pacotes de código.

A idéia é ter todo o código modularizado no JS.

Possui espaço para plugins e loaders.

Tudo no Webpack é configurado num arquivo webpack.config.js - nome padrão.


Resumindo:Seria algo como um empacotador de módulos estátios para aplicações modernas de Javascript. Ele vai pegar o seu arquivo de entrada e a partir dele, vai buscar todas as dependências e ao final gerar um arquivo contendo tudo em um único pacote, que geralmente salvamos na pasta dist.
